# Weather App

## Project setup

```
yarn install
```

## Available Scripts

### Compiles and hot-reloads for development

```
yarn start
```

### Compiles and minifies for production

```
yarn build
```

### Lints files

```
yarn lint
```

### Fix lint errors

```
yarn lint:fix
```
